Anatomy of the Tool

There are currently 3 main areas of the tool. The stats along the top, the d20 scores in the middle, and the level / rank / role (LRR) selectors at the bottom. Each stat has a corresponding d20 associated with it which represents the stat‚Äôs score from 1-20 on how strong it is for the selected LRR. All 10s would be a balanced typical statblock for the LRR. A nat 1 or 20 is pretty far outside normal range for a given LRR, and should usually be avoided.

Each stat can be changed either by its value directly, or by its score using the slider. You can double click on the d20 to reset to 10. The inputs to the calculator are selected by clicking on them, and are indicated in purple. The purple values will stay the same as you change LRR, while the grey values will be updated.

Converting a 5e Creature

There are more tools in the works to make this an easier process (like being able to enter 5e stats directly), but here‚Äôs a start. Let‚Äôs say we want to convert a Minotaur because there aren‚Äôt enough cows in DC20 yet. First we have to ballpark some conversions. This doesn‚Äôt have to be super exact because the tool will help us. This is what I use to get started.

  • HP: HP / 3
  • PD: AC
  • MD: 10 + Avg(Int Save, Wis Save, Cha Save)
  • Atk: Hit bonus
  • Dmg: Average attack damage / 4

Filling that in for our Minotaur, we get HP 25, PD 14, MD 10, Atk 6, Dmg 4. For a Level 3 High-Tier Brute, this doesn‚Äôt look too bad. With both higher than typical Atk and Dmg, but lower that typical HP. This is looking like a glass canon type monster, if that‚Äôs not the feel we want we can adjust accordingly.
